# 🚀 Desafio de Projeto: Abstraindo um Bootcamp com Orientação a Objetos em Java

Este projeto foi desenvolvido como parte de um **desafio prático de Programação Orientada a Objetos (POO)** utilizando **Java**.
O objetivo é aplicar na prática os conceitos fundamentais de POO, modelando um sistema que representa um **Bootcamp**, com **cursos, mentorias e desenvolvedores**.

---

# 📚 Objetivo do Desafio

O objetivo deste desafio é permitir que o desenvolvedor aprenda **na prática** os principais pilares da **Programação Orientada a Objetos**, utilizando exemplos desenvolvidos em **Java**.

Durante o projeto, são aplicados conceitos como:

* Abstração
* Encapsulamento
* Herança
* Polimorfismo

Esses conceitos são utilizados para modelar as entidades do sistema e simular a participação de desenvolvedores em um bootcamp.

---

# 🧠 Conceitos de Programação Orientada a Objetos

## 📌 Paradigma

* **Paradigma**: Um modelo ou padrão a ser seguido no desenvolvimento de software.

## 📌 Programação

* **Programação**: Processo de desenvolvimento de aplicações por meio de algoritmos e instruções.

## 📌 Orientada

* Algo que segue uma determinada orientação ou conjunto de regras.

## 📌 Objeto

* Um **objeto** é uma instância de uma **classe**, representando algo do mundo real dentro do sistema.

### 💡 Paradigma de Programação Orientado a Objetos

A **POO** é um modelo de desenvolvimento de aplicações baseado no conceito de **objetos**, que representam entidades do mundo real contendo **atributos** e **métodos**.

---

# ⚙️ Linguagem vs Paradigma

## 🖥️ Linguagem de Programação

Uma **linguagem de programação** é uma linguagem formal que permite que programadores escrevam instruções, algoritmos e estruturas para criar programas.

As linguagens podem ser classificadas por:

* Nível de abstração
* Paradigma de programação
* Arquitetura de aplicação
* Tipo de execução

---

## 🧩 Paradigma de Linguagem de Programação

Um **paradigma de programação** é um conjunto de características que define como um programa pode ser estruturado e desenvolvido.

Cada paradigma oferece técnicas apropriadas para resolver diferentes tipos de problemas.

### Principais Paradigmas

#### 🔹 Paradigma Imperativo

* Programação Estruturada
* Programação Concorrente
* Programação Orientada a Objetos

#### 🔹 Paradigma Declarativo

* Programação Funcional
* Programação Lógica
